# Numbers pattern

## Problem

Imprima un patrón de números desde 1 hasta n.

## Input Format

• n: entero

## Constraints

• Puedes asumir que todos n serán enteros positivos.

## Samples

### Sample input 0
`2`
### Sample output 0

4 3 2 2 2 3 4 <br>
4 3 2 1 2 3 4 <br>
4 3 2 2 2 3 4 <br>
4 3 3 3 3 3 4 <br>
4 4 4 4 4 4 4 <br>
